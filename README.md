# server

Como iniciar server terraria.

Vá na pasta do terraria, clique com o botão direito em terraria na steam -> Propriedades -> Arquivos locais -> Navegar por arquivos locais, selecione terraria server.


Coloque o mundo .wld e o .bak na pasta C:\Users\SEUUSER\OneDrive\Documentos\My Games\Terraria\Worlds

Inicie o terrariaserver.exe , selecione o mundo, porta , numero max de player etc .

Certifique-se de estar no hamachi.
